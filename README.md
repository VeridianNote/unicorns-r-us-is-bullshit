# Unicorns R Us Is Bullshit

A single-page static site debunking online misinformation.

Built with [Docusaurus](https://docusaurus.io/).

## Development

```bash
npm install
npm start
```

## Deployment

Deploys to GitHub Pages automatically on push to `main`.
